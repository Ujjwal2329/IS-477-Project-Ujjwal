# Data Folder

This folder contains data used in the IS 477 coffee project.

## `raw/`

- `coffee_sales.csv`  
  - Source: Kaggle – Coffee Sales Dataset  
  - Description: Transaction-level coffee sales data with transaction id, date, time, store location, product, quantity, unit_price, and product category/type.  
  - License & terms: Refer to original Kaggle dataset page.

- `coffee_shop.csv`  
  - Source: Kaggle – Coffee Shop Dataset  
  - Description: Coffee transaction data including hour_of_day, payment type, amount (`money`), coffee_name, and time-of-day fields.  
  - License & terms: Refer to original Kaggle dataset page.

## `processed/`

This folder will contain cleaned and transformed versions of the data generated by notebooks or scripts (e.g., `coffee_sales_clean.csv`, `coffee_shop_clean.csv`).
